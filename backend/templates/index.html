CUDA Visual Lab

Live Mandelbrot Renderer

This lab runs a background CUDA render loop. Each frame assigns one GPU thread per pixel, writes a fresh BMP image, and refreshes that image in the browser.

Human Summary

Ready to run a live CUDA visual

Start the live renderer to generate Mandelbrot fractal frames on the GPU. The browser will keep refreshing the latest CUDA-generated image.

  • Each pixel is computed by a CUDA thread.
  • The backend runs a render loop and serves the newest frame to the browser.
  • This is a basic live visual generator intended to stay simple and easy to reason about.
